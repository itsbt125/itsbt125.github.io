Introduction

Hey, I'm Brandon (btea), a teenager who's been around computers my whole life. I'm into Linux, networking, and malware analysis. I've been running a homelab for a while now, basically playing sysadmin — and I've been focused on leveling up my C++ and frontend skills after spending most of my time in Python.

My hardware

I'll admit I've never really owned truly old hardware. My desktop rocks a 9800X3D and 9070 XT, which is already overkill for what I do — and somehow I got my CPU with 64GB of RAM in a Microcenter bundle. I've grown to like the laptop life again lately, so I run an HP gaming laptop with the NVIDIA dGPU disabled for Linux, though I did have to swap out the Realtek 8852CE WiFi card due to stability issues. On the server side, I put an NVMe hat on my Raspberry Pi 5 a while back, which serves this site!

What I'm doing right now

Over the past few months I've fallen deep into the cybersecurity rabbit hole. I recently built a project that enriches fail2ban logs with ranked geolocation and ISP data. I'm also approaching my 4-month anniversary of fully switching to Linux on all my machines. Teaching myself C++ has been slow but steady, and this site marks my first HTML written outside of school — put together after about an hour of reading up on it.
